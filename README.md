## Getting Started

Please Run 

```bash
npm install
```

to install any packages that was miss.

Then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
